# 🎨 小画师模拟器 —— 需求文档

> **项目名称**：Little Artist Simulator（小画师模拟器）  
> **类型**：文字模拟经营 / 养成类游戏  
> **灵感来源**：青椒模拟器、大厂模拟器  
> **版本**：v0.1（初版需求）  
> **日期**：2026-03-27

---

## 一、游戏概述

### 1.1 核心概念

玩家扮演一名刚入行的**自媒体二次元小画师**，从零开始经营自己的绘画事业。通过日常选择（画画、接稿、摸鱼、学习、社交等），在**画技提升、金钱收入、精神健康、粉丝增长**之间寻找平衡，最终达成不同的人生结局。

### 1.2 游戏风格

- 以**文字选择 + 数值模拟**为核心玩法
- 轻松幽默的叙事风格，穿插二次元圈子的梗和日常
- 像素风 / 简约可爱的 UI 风格
- 单局游戏时长：30~60 分钟

## 二、核心属性系统

### 2.1 主要属性

| 属性 | 英文标识 | 初始值 | 范围 | 说明 |
|------|---------|--------|------|------|
| 💰 金钱 | `money` | 5000 | 0 ~ ∞ | 生存的基础，每月有固定支出（房租、生活费） |
| 🎨 画技 | `skill` | 10 | 0 ~ 100 | 决定作品质量、接稿价格、粉丝评价 |
| 🧠 精神状态 | `mental` | 80 | 0 ~ 100 | 过低会触发"摆烂"、"emo"等负面事件 |
| ❤️ 体力 | `stamina` | 100 | 0 ~ 100 | 每个行动消耗体力，睡觉/休息恢复 |
| 🌟 灵感 | `inspiration` | 50 | 0 ~ 100 | 影响作品创意度，高灵感可触发"神作"事件 |
| 👥 粉丝数 | `fans` | 0 | 0 ~ ∞ | 社交媒体粉丝量，影响约稿量和收入 |
| 🔥 热度 | `popularity` | 0 | 0 ~ 100 | 当前在圈内的热度，会随时间自然衰减 |
| 😊 社交值 | `social` | 30 | 0 ~ 100 | 与其他画师/甲方的关系网络 |

### 2.2 隐藏属性

| 属性 | 说明 |
|------|------|
| 画风倾向 | 根据玩家选择逐渐形成（萌系/厚涂/赛璐璐/写实等） |
| 口碑 | 影响甲方主动约稿概率和稿费 |
| 拖稿指数 | 拖稿次数越多，口碑越差 |
| 黑粉数 | 热度越高越容易积累，可能触发网暴事件 |

## 三、时间系统

### 3.1 基本结构

- 游戏以**周**为基本单位，每周分为 **上午 / 下午 / 晚上** 三个时间段
- 每个时间段可执行**一个主要行动**
- 游戏总时长为 **1年（365天）**，按月推进剧情节点
- 每月月底结算：扣除固定支出（房租 + 生活费）

### 3.2 特殊时间节点

| 时间 | 事件 |
|------|------|
| 每月1日 | 月度结算（房租、生活费扣除） |
| 春节期间 | 红包收入 / 亲戚灵魂拷问事件 |
| 暑假 | 漫展季，可选择参加漫展摆摊 |
| 年底 | 年度总结，触发结局判定 |

## 五、事件系统

### 5.1 随机事件

游戏中会根据当前属性状态随机触发各类事件，玩家需要做出选择：

#### 正面事件示例

| 事件 | 触发条件 | 选项 |
|------|---------|------|
| 🎉 作品突然爆了！ | 热度>50，画技>40 | A. 趁热打铁继续产出 B. 低调发展 |
| 💌 大V转发了你的图 | 粉丝>1000 | A. 感谢互动 B. 借机宣传约稿 |
| 📧 游戏公司邀请合作 | 画技>60，粉丝>5000 | A. 接受（高收入但时间紧） B. 婉拒 |
| 🏆 作品入选画集 | 画技>70 | A. 同意收录 B. 要求更高稿费 |
| 🎁 粉丝送了数位板 | 粉丝>3000，社交>50 | 装备升级，画技效率+10% |

#### 负面事件示例

| 事件 | 触发条件 | 选项 |
|------|---------|------|
| 😤 甲方要求大改 | 接了商业稿 | A. 忍气吞声改（精神-20） B. 据理力争 C. 退稿（口碑-） |
| 🗑️ 作品被盗图 | 粉丝>2000 | A. 维权（消耗时间精力） B. 发微博控诉 C. 算了 |
| 💀 被网暴了 | 黑粉数>阈值 | A. 正面回应 B. 关闭评论躲几天 C. 发长文回应 |
| 🤒 生病了 | 体力<20 且连续加班 | 强制休息2天，金钱-500（医药费） |
| 📉 创作瓶颈 | 灵感<10 | 画技暂时无法提升，需要寻找灵感 |
| 💸 房租涨了 | 随机 | 每月支出增加 |
| 🐦 发了争议言论 | 随机 | A. 道歉删除 B. 坚持己见 C. 装死 |

### 5.2 剧情事件（按时间线触发）

| 月份 | 事件 | 说明 |
|------|------|------|
| 第1月 | 开局：选择起点 | 选择初始画风、平台、设备 |
| 第3月 | 第一次约稿 | 引导玩家体验接稿流程 |
| 第5月 | 画师圈撕逼事件 | 选择站队或中立，影响社交关系 |
| 第7月 | 夏日漫展 | 是否参加漫展摆摊 |
| 第9月 | 瓶颈期 | 强制触发创作瓶颈，考验玩家应对 |
| 第11月 | 年终机遇 | 大型合作机会，根据全年积累决定是否能接 |
| 第12月 | 年度总结 | 结局判定 |

## 七、成长系统

### 7.1 画技等级

| 等级 | 画技范围 | 称号 | 解锁内容 |
|------|---------|------|---------|
| Lv.1 | 0~15 | 涂鸦小白 | 只能画火柴人 |
| Lv.2 | 16~30 | 入门画手 | 可以接低价约稿 |
| Lv.3 | 31~50 | 进阶画师 | 解锁商业稿件 |
| Lv.4 | 51~70 | 实力画师 | 解锁高价约稿、教程制作 |
| Lv.5 | 71~85 | 大触 | 解锁游戏/动画公司合作 |
| Lv.6 | 86~100 | 神仙画师 | 解锁顶级合作、出版画集 |

### 7.2 画风系统

玩家在游戏过程中的选择会逐渐形成画风倾向：

- **萌系**：粉丝增长快，约稿多但单价偏低
- **厚涂**：作品质感高，商业稿件多
- **赛璐璐**：适合动画相关合作
- **写实**：小众但高端，单价高
- **像素风**：独立游戏合作机会多

### 7.3 技能树（可选解锁）

| 技能 | 前置条件 | 效果 |
|------|---------|------|
| Live2D | 画技>40 | 解锁虚拟主播皮肤制作 |
| 动画制作 | 画技>50 | 解锁动画短片创作 |
| 3D建模 | 画技>45 | 解锁手办原型设计 |
| 漫画分镜 | 画技>35 | 解锁连载漫画 |
| UI设计 | 画技>30 | 解锁兼职UI设计师 |

```typescript
interface GameState {
  // 基础信息

  money: number;

interface Choice {

  effects: Effect[];          // 属性变化
```

## 十二、特色玩法亮点

1. **真实感**：融入大量真实画师圈的梗和日常（甲方修改、催稿、盗图维权等）
2. **选择的重量**：每个选择都有代价，没有完美解法，模拟真实的取舍
3. **多结局**：15+ 种结局，鼓励多周目游玩
4. **社交媒体模拟**：模拟发图、互动、涨粉的真实体验
5. **情绪共鸣**：精神状态系统让玩家体会画师的心理压力
6. **幽默文案**：大量吐槽向文案，让玩家会心一笑
